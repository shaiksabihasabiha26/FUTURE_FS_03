# Glow Beauty Salon Website – Task 3

## 📌 Project Overview
This project was developed as part of the Full Stack Web Development Internship at Future Interns.

The objective was to build a professional website for a local business and present it as a digital solution proposal.

---

## 🏢 Business Context
Glow Beauty Salon (Vijayawada)

Many small local businesses lack a strong online presence. This website demonstrates how a modern, responsive website can improve customer visibility and brand trust.

---

## 💻 Tech Stack
- HTML5
- CSS3
- Responsive Web Design
- Deployed using Vercel

---

## ✨ Features
- Modern hero section
- Service listing with pricing
- Image gallery
- Contact information section
- Fully responsive layout

---

## 📈 Business Impact
- Improved digital presence
- Clear service showcase
- Better customer engagement
- Increased potential walk-ins
